Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Use this link for istallastion and setup for Visual Studio Code on Windows computers: https://code.visualstudio.com/docs/setup/windows

Console is a basic computer or monitor and keyboard that is connected to another computer, server, or a mainframe over a network. It is used to maintain or monitor the status of the network or computer. The fastest way to directly open the Console is to press Ctrl+Shift+J.
